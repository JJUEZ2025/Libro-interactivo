# Libro interactivo

Plantilla básica en HTML, CSS y JS para historias interactivas.

## Cómo verificar si tus cambios se aplicaron
1. Guarda tus archivos y abre la terminal en la carpeta del proyecto.
2. Ejecuta `git status` para ver qué archivos cambiaste y confirmar que Git los detecta.
3. Abre `index.html` en tu navegador (por ejemplo con una extensión de servidor local o usando `python -m http.server 8000`) y revisa visualmente los cambios.
4. Si quieres comparar con la versión anterior, usa `git diff` para ver las diferencias línea por línea.
5. Cuando estés conforme, ejecuta `git add .` y `git commit -m "mensaje"` para registrar tus cambios.

Estos pasos te permiten validar rápidamente que las modificaciones realizadas se guardaron y se ven como esperas.
